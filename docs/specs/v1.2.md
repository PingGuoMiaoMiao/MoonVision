# MoonVision v1.2 Spec

## Status

- Phase: approved
- Base branch: `v1.1`
- Goal: extend MoonVision from basic visual analysis into a broader lightweight `imgproc` layer through edge and contour analysis

## Summary

`v1.2` focuses on three deliverables:

- algorithm layer: `Canny`, contour extraction, contour statistics
- demo layer: upgraded `edge_detection` and `object_counting`
- docs layer: `README` updates with `v1.1` vs `v1.2` notes

## In Scope

1. `canny_edges(image, low_threshold, high_threshold)`
2. `find_contours(binary_image, connectivity?)`
3. `contour_area(contour)`
4. `contour_perimeter(contour)`
5. `contour_bbox(contour)`
6. `edge_detection` demo output for both Sobel magnitude and Canny
7. `object_counting` demo output with contour statistics
8. `README` updates for new APIs and outputs

## Constraints

- contour extraction only accepts binary `GrayImage`
- `connectivity` defaults to `8`
- no RGB contour input in this version
- no perspective transform, histogram enhancement, or DNN/video scope
- no high-level combined API that hides thresholding + contour extraction in one call

## Acceptance

- `moon check`
- `moon test`
- `moon run src/demo/edge_detection`
- `moon run src/demo/object_counting`
- `README` reflects `v1.2` APIs, outputs, and comparison notes
